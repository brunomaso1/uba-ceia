Bienvenido Nueva predicción

Realiza una nueva predicción de palmeras ingresando los datos necesarios. Previamente, deberías tener el ortomosaico y el archivo JGW de la zona de interés.
Al realizar una predicción, el sistema analizará la imagen y generará un resultado que podrás visualizar en la sección de resultados.
También se permitirá exportar los resultados a un KML (Keyhole Markup Language) para su uso en aplicaciones de mapas.
